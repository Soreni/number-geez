# number-geez-module

number-geez-module is a npm library for converting Arabic number to Ethiopian number (Geez).

## Installation

Use npm package manager  to install number-geez.

```bash
npm install number-geez
```

## Usage

```Node.js
var ntg = require('number-geez')

ntg(1) # returns ·ç©
```
